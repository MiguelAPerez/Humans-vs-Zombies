# RC Car Setup

### Modifications

1. Removing the top of the car 
2. Removing the cars microprocessor
3. Resoldering the on/off switch 


### Wireing 


    STBY = Pin 13 (GPIO #21)    = Stand By

    Drive motor:
    PWMA = Pin 7  (GPIO #4)      = Speed
    AIN2 = Pin 11 (GPIO #17)
    AIN1 = Pin 12 (GPIO #18)

    Left-Right controller:
    BIN1 = Pin 15 (GPIO #22)
    BIN2 = Pin 16 (GPIO #23)    
    PWMB = Pin 18 (GPIO #24)    = Speed
