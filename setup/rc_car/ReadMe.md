# RC Car Setup

### Modifications

1. Removing the top of the car 
2. Removing the cars microprocessor
3. Resoldering the on/off switch 


### Wiring 


    STBY = GPIO #17     = Stand By

    Drive motor:
    PWMA = GPIO #3      = Speed
    AIN2 = GPIO #4
    AIN1 = GPIO #18

    Left-Right controller:
    BIN1 = GPIO #27
    BIN2 = GPIO #23    
    PWMB = GPIO #24    = Speed
